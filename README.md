Final Project
Emotion Detection Application

This project is part of the IBM Developing AI Applications with Python and Flask course.

The application analyzes user-provided text and detects emotions using the Watson NLP Emotion Prediction service.

Features
Detects anger, disgust, fear, joy, and sadness
Identifies the dominant emotion
Provides error handling for blank input
Includes unit tests for emotion detection
Deploys a web application using Flask
Technologies Used
Python
Flask
Requests
Watson NLP Emotion Detection API
PyLint
Unit Testing
